# Fintech Product Growth Analysis

This project explores how early user activation influences retention and lifetime value.

Using transactional data, I analyzed whether users who engage early with a product demonstrate stronger retention patterns and higher long-term value.

Key findings:
- Only ~10% of users activate early
- Activated users retain significantly longer
- Activated users generate ~5x higher lifetime value

The analysis demonstrates how improving early activation could significantly increase product engagement and revenue.
